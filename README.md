# Athena Dashboard

A modern Business Intelligence (BI) web application for visualizing healthcare AI product data stored in Amazon S3 (Parquet format) and queried via Amazon Athena.

## Overview

The Athena Dashboard provides internal stakeholders (product, operations, leadership, analytics teams) with powerful analytics and visualization capabilities for healthcare data generated by an AI-based healthcare product.

## Getting Started

### Prerequisites

- Python 3.10+
- Node.js 18+ and pnpm
- AWS credentials with Athena access
- Access to S3 bucket with Parquet data

### Environment Setup

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd athena_dashboard
   ```

2. **Backend Setup**
   ```bash
   cd backend
   pip install poetry
   poetry install
   cp .env.example .env
   # Edit .env with your AWS credentials
   poetry run uvicorn app.main:app --reload
   ```

3. **Frontend Setup**
   ```bash
   cd frontend
   npm install
   cp .env.example .env.local
   # Edit .env.local if needed
   npm run  dev
   ```

### Environment Variables

**Backend (.env):**
```bash
AWS_ACCESS_KEY_ID=your_access_key
AWS_SECRET_ACCESS_KEY=your_secret_key
AWS_REGION=us-east-1
ATHENA_WORKGROUP=primary
ATHENA_DATABASE=healthcare_analytics
ATHENA_TABLE=patient_data
ATHENA_OUTPUT_S3=s3://athena-results/
JWT_SECRET_KEY=your-secret-key-here
```

**Frontend (.env.local):**
```bash
VITE_API_URL=http://localhost:8000
```
